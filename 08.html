CSS 기본

CSS 기본

  • Html 문서에서 디자인을 꾸미는 모든 활동

CSS 작성방식

  • 인라인방식 : 태그(요소)에 style 속성으로 작성하는 방식, 비권고방식
  • 내부스타일 : style 태그로 작성하는 방식, html 파일 안에 있으므로 비권고방식
  • 외부스타일 : link 태그로 작성하는 방식, 웹표준, 권고사항

CSS 선택자

    span
  • 스타일링 대상 요소(태그) 또는 그 경로
  • * : 전체선택자
  • 태그(li) : 요소선택자
  • # : id 선택자 => 고유한 이름을 줌, 유일무이함, 변화가 없는 경우
  • . : 클래스 선택자, 한개 이상의 특정 그룹, 여러개 줄 수 있음,변화가 많은 경우
  • (빈칸) : 자손선택자, 자손이라는 관계성격 부여
  • > : 자식선택자, 자식이라는 관계성격 부여
  • :nth-child(숫자) : 형제 번호가 숫자 몇번째인지의 요소
  • [속성] : 속성선택자
  • + : 형제선택자 => 앞이 input인 input에게 부여
  • ~ : 형제선택자 => 선택자의 동생들을 선택
  • , : 그룹선택자

    • CSS 색상표현

    • 색상명 : red
    • 웹 색상코드(16진수) : #ff0000, 0~9 + a~f, 약식표현 : #f00
    • RGB : (255,0,0)
    • RGBA : (255,0,0,1)

    CSS 단위

    • pt,cm,mm : 인쇄용, 비권고
    • px : 픽셀, 모니터의 점 한개, html 문서의 표준 단위
    • % : 부모 요소에 대한 배율 단위
    • rem : 루트(html,body 크기와 똑같이 쓰겠다)가 가진 글자크기에 대한 배율단위
    • em : 부모가 가진 글자크기에 대한 배율단위
    • 숫자 : x배
    • vh, vw : 뷰포트의 배율 단위, %와 비슷함
