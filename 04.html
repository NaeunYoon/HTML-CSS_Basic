웹 표준 마크업

  • 의미에 맞는 요소(태그)를 선택하여 사용 => 시멘틱 마크업
  • 웹의 3요소 : 구조 (html, contens), 디자인(css), 동작(javascript)의 파일분리
  • 퍼블리셔 : 웹 디자인(구조와 디자인), 프론트엔드 : 동작
  • 시멘틱 태그를 적극적으로 활용
  • 콘텐츠에는 항상 제목을 달아주는 것을 권장
  • 사용자 입력폼 (폼문) 은 form, filedset, legend는 같이 사용한다
  • 디자인적인 요소는 생략, 기능이나 컨텐츠, 내용중심으로 작업
  • 링크는 a, 동작은 btn
  • 유효성검사 html validator 마크업 문법 검사는 필수
  • url=>사이트 주소, uri => 사이트 풀 주소
  • 의미가 없는 디자인용 태그 div, span
  • 비권장태그 : 디자인적인 의미를 가지는 태그 (b,i,u,center,hr...)
  • 프레인관련태그 : frameset, frame, iframe
